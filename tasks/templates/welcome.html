API Endpoints

Tasks
  • GET /api/tasks/ - List all tasks
  • POST /api/tasks/ - Create a new task
  • GET /api/tasks/{id}/ - Get task details
  • PUT /api/tasks/{id}/ - Update task
  • DELETE /api/tasks/{id}/ - Delete task
Task Assignments
  • POST /api/tasks/{id}/assign/ - Assign users to a task
  • GET /api/tasks/{id}/assignments/ - Get task assignments
Users
  • GET /api/users/ - List all users
  • POST /api/users/ - Create a new user
  • GET /api/users/{id}/ - Get user details
  • PUT /api/users/{id}/ - Update user
  • DELETE /api/users/{id}/ - Delete user
  • POST /api/users/{id}/change_password/ - Change user password
  • POST /api/users/{id}/reset_password/ - Reset user password
  • GET /api/users/me/ - Get current user profile
  • PUT /api/users/me/ - Update current user profile
User Tasks
  • GET /api/users/{id}/tasks/ - Get tasks assigned to a user
Authentication
  • POST /api/token/ - Get authentication token
  • POST /api/token/refresh/ - Refresh authentication token

Getting Started

  1. Access the Admin Interface to manage users and tasks
  2. View the API Documentation for detailed endpoint information
  3. Use the API endpoints with proper authentication
