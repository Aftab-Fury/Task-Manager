Create, assign, and track tasks with ease. Support for multiple task types and statuses.

API Documentation

Comprehensive API documentation with Swagger UI for easy integration.

API Endpoints

Tasks:
  • GET /api/tasks/ - List all tasks
  • POST /api/tasks/ - Create a new task
  • GET /api/tasks/{id}/ - Get task details
  • PUT /api/tasks/{id}/ - Update task
  • DELETE /api/tasks/{id}/ - Delete task
Task Assignments:
  • POST /api/tasks/{id}/assign/ - Assign task to users
  • GET /api/tasks/{id}/assignments/ - Get task assignments
User Tasks:
  • GET /api/users/{id}/tasks/ - Get tasks assigned to a user

Getting Started

  1. Access the Admin Interface to manage users and tasks
  2. View the API Documentation for detailed endpoint information
  3. Use the API endpoints with proper authentication
